# ITSM Help Desk Integration (Jira Cloud & Automation)

To simulate a fast-paced enterprise IT environment, I established a cloud-hosted Jira Service Management sandbox and integrated it with an automation pipeline to handle incoming technical incidents.

## Technical Achievements:
* **API Ingestion Engine:** Developed a custom PowerShell automation script to programmatically inject simulated user tickets into the Jira Service Management core via a secure Atlassian REST API gateway.
* **Incident Lifecycle Management:** Triaged and processed automated tickets mapped directly to cross-departmental users (Human Resources, Finance, and Operations) provisioned during Phase 1.
* **Enterprise Tier Escalation:** Engineered a structured Tier-1 to Tier-2 internal markdown escalation template to cleanly hand off deep access-control anomalies to senior engineering teams.

## Operational Scenarios Managed:
1. **Account Lockout Recovery:** Performed administrative credential overrides inside Microsoft Entra ID and enforced a mandatory password rotation loop upon the client's next login phase.
2. **Asset Management Allocation:** Logged physical hardware matrix damage, tracked asset swap logs for replacement inventory, and updated internal provisioning records.
3. **Permission Hierarchy Escalation:** Diagnosed file-share access loops, completed foundational baseline troubleshooting, and executed formal cross-team escalation.
